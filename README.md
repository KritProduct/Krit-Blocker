Блокировщик приложений для Windows с проверкой SHA256-хешей, историей названий процессов и расширенными режимами маскировки.

---

## Возможности

### Блокировка процессов
- Блокировка любых .exe приложений по названию
- Проверка по SHA256-хешу (невозможно обойти переименованием файла)
- История названий (запоминает все имена заблокированных программ)
- Защита от блокировки системных процессов

### Маскировка и скрытность
- Смена названия окна на любое (маскировка под системный процесс)
- Скрытие из Диспетчера задач
- Сворачивание в системный трей
- Вход и выход только по паролю

### Дополнительная защита
- Блокировка CMD, PowerShell, Диспетчера задач, RegEdit
- Режим помех для AnyDesk (создаёт лаги через высокоточный таймер)

### Интерфейс
- Два языка: Русский и English
- Тёмная тема оформления
- 12 режимов ошибок (включая фейковый BSOD)

### Пароль
- Создание пароля при первом запуске
- SHA256-хеширование
- Возможность смены пароля
- Кнопка "Показать пароль"
